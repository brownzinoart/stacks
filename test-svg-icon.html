iOS app icon design for the Stacks library book discovery app.

Design Notes

  • Theme: Stack of books representing library discovery
  • Colors: Blue gradient background with colorful book stack
  • iOS Compliant: Proper corner radius, no transparency
  • Scalable: Works well at all required iOS sizes
  • Professional: Clean, modern design suitable for App Store

Technical Specs

  • Background: Linear gradient (#1e40af to #0369a1)
  • Books: Orange (#f59e0b), Green (#10b981), Red (#ef4444), Purple (#8b5cf6)
  • Corner radius: 22% of icon size (iOS standard)
  • Drop shadow for depth
  • Subtle highlight on top book
